# maker-week-interactive-art

Run the WebSocket proxy:
```
cd websocket-proxy
npm install
node main.js
```

Build and open the video UI:
```
cd hand-detection
npm install
npm run watch
npm run watch
```

Build the code for the drawing plugin:
```
cd drawing-plugin
npm install
npm run build
```

Load the drawing plugin into a Figma desktop app, then run it
